模块方法
方法 描述 参数 返回值
on 监听事件
(string：监听类型[(过滤器)])
多个监听类型可用空格分隔
trigger 触发事件
(string：监听类型[(过滤器)]，object：回调函数的参数))
多个监听类型可用空格分隔
render 动态生成菜单
(object：参数配置)
 菜单实例
实例方法
方法 描述 参数 返回值
on 监听事件
(string：监听类型)
多个监听类型可用空格分隔
实例方法只能监听该实例对应的菜单事件
trigger 触发事件
(string：监听类型，object：回调函数的参数))
多个监听类型可用空格分隔
实例方法只能触发该实例对应的菜单事件
getChecked 获取已选中的数据 array
spread 展开某一项 (id：唯一标识)
close 关闭某一项 (id：唯一标识)
reload 重载实例 (string：和 render 参数一致)
render 参数
参数 类型 描述 默认值
elem string/dom
外部容器
 document.body
data array
左侧数据列表，结构：{id：唯一值，title：标签名称，disabled：是否禁用，spread：是否默认展开，children：子树}
spread boolean
是否默认都展开
showCheckbox boolean
是否显示多选框
 false
style string
树形结构风格，可选值："arrow"、"line"
 "line"
width number
树形结构宽度
height number
树形结构高度
onlyIconSwitch boolean
是否只允许icon来控制展开/收起
 false
search boolean
是否支持搜索
 false
searchMethod function
自定义搜索函数
事件
事件 描述 回调参数
click 标题文本点击事件 {dom：触发 dom，data：当前项的数据}
change 多选框事件 {dom：触发 dom，data：当前项的数据，checked：是否选中}
spread 展开事件 {dom：触发 dom，data：当前项的数据}
close 收起事件 {dom：触发 dom，data：当前项的数据}
