模块方法
方法 描述 参数 返回值
on 监听事件
(string：监听类型[(过滤器)])
多个监听类型可用空格分隔
trigger 触发事件
(string：监听类型[(过滤器)]，object：回调函数的参数))
多个监听类型可用空格分隔
render 动态生成菜单
(object：参数配置)
 菜单实例
实例方法
方法 描述 参数 返回值
on 监听事件
(string：监听类型)
多个监听类型可用空格分隔
实例方法只能监听该实例对应的菜单事件
trigger 触发事件
(string：监听类型，object：回调函数的参数))
多个监听类型可用空格分隔
实例方法只能触发该实例对应的菜单事件
getData 获取已移动的数据 array
reload 重载实例 (string：和 render 参数一致)
render 参数
参数 类型 描述 默认值
elem string/dom
外部容器
 document.body
data array
左侧数据列表，结构：{id：唯一值，title：标签名称，checked：是否选中，disabled：是否禁用}
value array
右侧数据列表，结构和 data 一致
stretch boolean
穿梭框是否铺满容器
 false
width number
穿梭框宽度
 200
height number
穿梭框高度
 320
leftTitle string
左侧穿梭框标题
rightTitle string
右侧穿梭框标题
search boolean
是否支持搜索
 false
searchMethod function
自定义搜索函数
事件
事件 描述 回调参数
change 穿梭框移动时触发 {dom：触发按钮，data：已移动的数据列表}
